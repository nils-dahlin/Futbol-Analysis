# Football Stats Analysis (FBref Scraper)

## Overview
This project is a Python-based data analysis tool that scrapes football (soccer) statistics from **FBref** and analyzes performance data for **Premier League teams and players** over the last few seasons.

The goal is to collect, clean, and analyze data such as:
- Team performance metrics
- Player statistics
- Season-by-season comparisons
- Trends and insights across multiple seasons
